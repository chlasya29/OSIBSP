# Oasis Infobyte Web Development Internship (OIBSIP)

## 📌 Overview

This repository contains the projects completed during my **Oasis Infobyte Web Development Internship (AICTE SIP)**. These projects were developed to strengthen my understanding of frontend web development using **HTML, CSS, and JavaScript**.

---

## 🚀 Projects

### 📱 Level 2 - Task 1: Calculator

A responsive calculator capable of performing basic arithmetic operations such as addition, subtraction, multiplication, and division. The project provides an interactive interface with real-time calculations.

**Features**
- Basic arithmetic operations
- Responsive design
- User-friendly interface

**Technologies Used**
- HTML5
- CSS3
- JavaScript

---

### 🎨 Level 2 - Task 2: Tribute Page

A tribute webpage dedicated to **Raja Ravi Varma**, showcasing his life, achievements, and contributions to Indian art through an attractive and responsive layout.

**Features**
- Informative content
- Images and structured sections
- Responsive design

**Technologies Used**
- HTML5
- CSS3

---

### ✅ Level 2 - Task 3: To-Do Web Application

A task management web application that allows users to efficiently organize their daily activities.

**Features**
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Separate Pending and Completed task lists
- Responsive interface

**Technologies Used**
- HTML5
- CSS3
- JavaScript

---

### 🔐 Level 2 - Task 4: Login Authentication System

A simple login authentication system that enables users to register, log in, and access a secured page.

**Features**
- User Registration
- User Login
- Authentication Validation
- Protected Page Access
- Responsive Design

**Technologies Used**
- HTML5
- CSS3
- JavaScript
